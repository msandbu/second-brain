# Published books

Five books published through Packt Publishing. All available on Amazon. Author page: [amazon.com/author/mariussandbu](https://www.amazon.com/Marius-Sandbu/e/B0BXVLHDTL)

---

## Windows Ransomware Detection and Protection

**Published:** 2023  
**Publisher:** Packt Publishing  
**Amazon:** [View on Amazon](https://www.amazon.com/dp/B0BNC1KF7L/)

A practical guide to securing Windows endpoints, cloud workloads, and on-premises infrastructure against ransomware using the Microsoft security stack. Covers Microsoft Intune for endpoint management and compliance, Microsoft Sentinel for threat detection and response, and Microsoft Defender for layered protection across environments.

The through-line: real ransomware defence is not about any single product but about the combination of endpoint management, threat detection, and response capability working together. Written from actual enterprise deployment experience.

Key topics: Intune device compliance and conditional access, Sentinel analytics rules and incident response, Defender for Endpoint and Defender for Cloud, attack surface reduction, backup and recovery strategy.

---

## Microsoft System Center Configuration Manager: High Availability and Performance Tuning

**Publisher:** Packt Publishing  
**Amazon:** [View on Amazon](https://www.amazon.com/dp/B00GX67TPO/)

A deep-dive into SCCM architecture for enterprise environments where availability and performance are non-negotiable. Covers site design for high availability, database and SQL Server optimisation, distribution point strategy, and performance tuning for large-scale deployments.

Written when SCCM was the dominant enterprise endpoint management platform, before the shift to cloud-based management with Intune. Still relevant for organisations with large on-premises SCCM deployments.

---

## Implementing NetScaler VPX (Second Edition)

**Published:** 2015  
**Publisher:** Packt Publishing  
**Amazon:** [View on Amazon](https://www.amazon.com/dp/B015CYBPSO/)

Updated and expanded guide to deploying and configuring Citrix NetScaler VPX in virtualised environments. Covers load balancing, SSL offloading, content switching, authentication, and integration with Citrix XenApp and XenDesktop.

NetScaler (now Citrix ADC) was a critical component in enterprise application delivery architectures. This edition reflects the changes in the product and the shift toward more complex multi-datacenter deployments.

---

## Implementing NetScaler VPX (First Edition)

**Published:** 2014  
**Publisher:** Packt Publishing  
**Amazon:** [View on Amazon](https://www.amazon.com/dp/178217267X/)

The original hands-on guide to NetScaler VPX deployment. Practical, architecture-focused, based on real deployment experience rather than product documentation. One of the first practical books on NetScaler VPX available at the time.

Covers initial setup, virtual server configuration, SSL, authentication, and integration with the Citrix stack.

---

## A note on approach

All five books follow the same pattern: practical over theoretical, based on real deployments, written to help people who are stuck on the same problems Marius has encountered in the field. No padding, no vendor marketing dressed up as technical content.
